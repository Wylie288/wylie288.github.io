Ultimate Chess

Ultimate Chess is an new take on chess. Featuring new pieces, a larger board, and the ability to place your own pieces before the match starts

Technical Information

Project Details

IDE: Unity/Visual Studio
Languages: C#
Stage: Final

Notable Features

  • Before the match starts players place their pieces until they reach a maximum value
  • When selecting a piece to move the game shows you valid positions as well as what opponents pieces it can take
  • check detection
  • The game manages turn order
  • Save game/Load game system
  • A system designed for quickly adding new pieces and defining their behaviour


    • What I learned

    This project offered a few new experiances. Ultimate Chess was the first time I used unity or C# beyond following a simple youtube tutorial. Ultimate Chess was also the first time I worked with others on a project's code base. The most important lesson I learned with this project was what coding paradigms were. The CS class I had taken this semester taught us C++ and functional design. I laid the groundwork for this project using functional design under the assumption the object oriented paradigm I have been using in Gamemaker was seen as "the easy/unproffesional way". It wasn't until a senior had noticed and asked why we weren't taking better advantage of C#s inheritence that I learned about the concept of paradigms.
